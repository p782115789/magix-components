mx-calendar

默认情形

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"></div>

默认选中某天

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-selected="2018-01-01"
></div>

设置可选择的范围

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-min="2017-08-05"
    view-max="2017-08-22"
    view-selected="2017-08-12"
></div>

设置周二为一周的开始

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-week-start="<%@ 2 %>"
></div>

带时分秒

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-time-types="all"
></div>

带时分秒的选中

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-time-types="all"
    view-selected="2018-08-08 18:08:20"
></div>

带时分秒,只能选择小时和分钟

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-time-types="hour,minute"
></div>

change事件

HTML Code:
<div mx-view="app/gallery/mx-calendar/index"
    view-time-types="all"
    view-selected="2018-08-08 18:08:20"
    mx-change="showDatetime()"
></div>
Javascript Code:
let Magix = require('magix');
let GTip = require('mx-gtip/index');
module.exports = Magix.View.extend({
    tmpl: '@index.html',
    mixins: [GTip],
    render() {
        let me = this;
        me.updater.digest();
    },
    'showDatetime<change>' (e) {
        this.gtipRT('日期：' + e.date + '，时间：' + e.time);
    }
});
